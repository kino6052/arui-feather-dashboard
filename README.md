ARUI Feather Stub
=================

Как запустить? (разработческий режим)
-------------------------------------

1. `npm i`
2. `npm run start`

Сборка проекта в production режиме
----------------------------------

1. `npm i`
2. `npm run build`
3. После чего можно проверить сборку командой `NODE_ENV=production ./.build/server.js`
